# About :

<b>This is a  Booking Appointment Booking System with Login, Sign up functionality and also having service page,  appointment booking page and who are very experienced.</b>

<br>

## Deployed Link :

   - Frontend: <a href="https://dentcare24x7.netlify.app/">Netlify</a>
   - Backend: <a href="https://dent-care-backend-aa29.onrender.com">Render</a>


<br>

# Requirements : 

- User can login and sign up
    - OTP will be sent by nodmailer
- User can login with google account
- User can visit pages 
- User can see services 
- User can book services / appointments and pay
    - User can view appointment 
    - User rescheduled apointments
    - User can cancel appointment
    - User can book the appointment for someone else.
    - User can give feedback on appointment
    - User can see account details / update them / get previous appointment  
    - User will be notified confirmationi email
- Feedback 
- Logout 

<br>

# Additional Requirements :
- We can give home services

<br>

# Schema : 

- User 
     - name
     - email
     - password

- Dentist 
     - name 
     - email 
     - salary
     - image

- Patient 
     - name 
     - email 
     - salary
     - image

- Services 
    - service_name 
    - service_image
    - service_price
    - service_description
    - service_category 
    - service_by_gender

- Slots 
     - slotID
     - date
     - time


- appointments 
    - stylerid
    - userID
    - serviceId
    - date,
    - time,
    - service_name,
    - service_des,
    - styler_name

## Run Locally

Clone the project

```bash
  git clone https://github.com/aniketbabariya24/tough-health-8180.git
```

Go to the project directory

```bash
  cd Backend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
